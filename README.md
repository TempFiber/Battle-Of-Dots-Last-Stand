BATTLE OF DOTS: LAST STAND
Overview

Battle Of Dots: Last Stand is a fast-paced real-time strategy game focused on tactical unit control, smart positioning, and quick decision-making. Instead of overwhelming you with complex systems, the game challenges your ability to think and react under pressure.

You command groups of units on a dynamic battlefield where every movement matters. Proper placement, timing, and coordination can mean the difference between victory and total defeat.

Core Gameplay
Control multiple units in real time
Strategically position your forces before and during battle
React quickly to enemy movements
Use formations and movement paths to outplay opponents
Manage generals and assign units for better control

The game blends elements of classic RTS mechanics with modern, streamlined controls, making it easy to learn but hard to master.

Features
- Fast-paced tactical combat
- Emphasis on positioning and decision-making
- Advanced unit selection and control system
- General-based unit grouping
- Pre-battle placement phase
- Line and curve movement commands
- Smooth camera zoom and battlefield overview


WASD - move the character

Mouse scroll - zoom camera in and out

Unit Selection

Left mouse click - select a single unit

Left mouse click + drag - select multiple units

Left click on general - select units in general

Unit Commands

Right mouse click - send or place units in placement mode

Right mouse click + drag - send units along the line you draw

Right mouse click + drag + Ctrl - send units along the curve line you draw

H - stop selected units

Formation & Placement

Shift + Right mouse click (in pre-battle placement mode) - arrange units in a line

Backspace - delete selected units in the placement phase

General Management

Ctrl + Right click on general - assign selected units to general

G - unassign selected units from general

Interface

Tab - Menu

Escape - deselect units


About the Project:

Battle Of Dots: Last Stand is designed as a minimalistic yet deep strategy experience. The goal is to create gameplay that is easy to pick up, but rewards skill, precision, and tactical thinking.
